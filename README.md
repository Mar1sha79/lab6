Структура проекта:
Interfaces.cs - содержит все интерфейсы проекта
Cat.cs - класс, представляющий кота с возможностью мяукать
Fraction.cs - класс для работы с математическими дробями
CachedFraction.cs - класс дроби с кэшированием вещественного значения
MeowHelper.cs - вспомогательные методы для работы с мяукающими объектами
Program.cs - главный класс программы с пользовательским интерфейсом

Основные методы
Класс Cat (Cat.cs)
Конструктор public Cat(string name) - создание кота с указанным именем
Метод Meow() - издание мяукающего звука один раз
Метод Meow(int count) - издание мяукающего звука несколько раз
Свойство MeowCount - получение количества выполненных мяуканий
Метод ToString() - возврат строкового представления в формате "кот: Имя"
Класс Fraction (Fraction.cs)
Конструктор public Fraction(int numerator, int denominator = 1) - создание дроби

Методы операций:
Add(Fraction other) - сложение с другой дробью
Subtract(Fraction other) - вычитание другой дроби
Multiply(Fraction other) - умножение на другую дробь
Divide(Fraction other) - деление на другую дробь
Перегруженные операторы: +, -, *, / для работы с дробями и целыми числами
Метод Clone() - создание копии дроби
Свойство RealValue - получение вещественного значения дроби
Конструктор public CachedFraction(int numerator, int denominator = 1)
Метод MakeAllMeow<T>(IEnumerable<T> meowables) - вызывает мяуканье у каждого объекта в коллекции
Метод MakeAllMeowAndCount<T>(IEnumerable<T> meowables) - вызывает мяуканье и возвращает общее количество мяуканий
Метод Main() - точка входа в программу
Метод RunInteractiveMode() - запуск интерактивного режима
Метод InteractiveCats() - работа с котами через консольный интерфейс
Метод InteractiveFractions() - работа с дробями через консольный интерфейс

Интерфейсы (Interfaces.cs)
void Meow() - издание мяукающего звука
void Meow(int count) - издание мяукающего звука несколько раз
int MeowCount { get; } - количество выполненных мяуканий
T Clone() - создание копии объекта
double RealValue { get; } - получение вещественного значения

Интерактивный режим
Работа с котами:
РАБОТА С КОТАМИ

Доступные действия:
1 - Создать кота
2 - Показать всех котов
3 - Заставить кота мяукать
4 - Заставить всех котов мяукать
5 - Показать статистику мяуканий
0 - Назад

Работа с дробями:
РАБОТА С ДРОБЯМИ

Доступные действия:
1 - Создать дробь
2 - Показать все дроби
3 - Выполнить операцию
4 - Сравнить дроби
5 - Клонировать дробь
6 - Показать вещественное значение
0 - Назад
